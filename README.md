# LEITURA DE CARTEIRINHA
Código arduino para **ler o UID de uma carteirinha** utilizando um sensor NFC/RFID, e mostrar no display se a carteirinha foi detectada.
- Sensor utilizado: NFC Module V3
- Display utilizado: ESP32-S3 4.3 inch 800*400 IPS with Touch
- Bibliotecas utilizadas: Arduino GFX Library (para o display), Adafruit PN532 (para o sensor) e lvgl (para criar a interface).
